Momentum (7D Return)

M = ((P_t - P_{t-7}) / P_{t-7}) * 100

Risk (Volatility, Coefficient of Variation)

mu = (1 / n) * Sigma(P_i)
sigma = sqrt((1 / n) * Sigma((P_i - mu)^2))
Vol_pct = (sigma / mu) * 100

Liquidity

L = min(100, (1 / n) * Sigma(V_i) / 7)

Mean-Reversion

z = (P_t - mu_7d) / sigma_7d
MR = clamp(-100, 100, -10 * z)

Composite Score

Score = 0.45*M + 0.20*(100 - Vol_pct) + 0.20*L + 0.10*MR + 0.05*RarityBonus

Confidence

Conf = clamp(0.1, 0.99, (window_len/10) * (1 - Vol_pct/100))
